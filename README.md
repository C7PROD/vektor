# VEKTOR — AI Voice Intelligence Agent

> Multi-mode AI voice agent for Star Citizen, built on ElevenLabs Conversational AI and Claude.

**Live demo:** [class7prod.com/vektor](https://www.class7prod.com/vektor)

---

## What It Does

VEKTOR is a voice-activated AI agent that delivers real-time game intelligence, tactical analysis, and military doctrine reference for a 100+ member Star Citizen military organization (8th ESG / 7th MCB).

Nine operational modes, all voice-activated, all available 24/7:

| Mode | Description |
|------|-------------|
| Doctrine & Training | Standing orders and operational procedures |
| Game Intelligence | Ships, systems, locations, mechanics |
| Loadout & Components | Component comparison, grade recommendations, sell-back pricing |
| Mining & Resources | Full Prospector/MOLE loop, ROC mining, cave harvestables, Rayari missions |
| Org OSINT | Intel on opposing organizations |
| Tactical Image Analysis | Screenshot and imagery interpretation |
| Military Strategy | Higher-order planning (Ops Access) |
| Guided AAR | After-action review structure |
| Guided OPORD | Operations order generation (Ops Access) |
